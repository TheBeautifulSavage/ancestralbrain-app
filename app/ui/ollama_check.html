Ancestral Brain uses Ollama to run AI locally on your machine. It's free, open-source, and your data never leaves.

Step 1 — Install Ollama
brew install ollama
Step 2 — Start it
ollama serve
Step 3 — Pull the AI models (one-time, ~8GB)
ollama pull llama3.1:8b
ollama pull nomic-embed-text

Already have Ollama? Make sure ollama serve is running, then click Check Again.
